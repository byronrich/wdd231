A place to grow, learn, and share the joy of gardening

Site Name

Garden Collective – This name reflects a shared space for gardening enthusiasts to explore, learn, and connect. It emphasizes community and the collective joy of growing plants.

Optional domain availability: gardencollective.org

Site Purpose

The site serves as a hub for beginner and intermediate gardeners. It offers:

  • A basic gardening guide
  • A curated list of popular yard plants
  • Personal insights and stories from the creator’s gardening journey

Scenarios

  • What are the easiest plants to grow in a backyard garden?
  • How do I start gardening if I’ve never planted anything before?

Color Schema

  • #3a6b35 (Forest Green) – Used for headings, accents, and footer text.
  • #f4f9f4 (Mint Cream) – Used for page background and section borders.

Typography

  • Montserrat – Used for all headings to create a clean, modern look.
  • Lora – Used for body text to provide a warm, readable style.

CSS

Styling is applied using the selected color schema and typography. The CSS file styles.css includes layout, spacing, and responsive design rules to support both mobile and desktop views.

Testing

  • HTML and CSS validated using W3C Validator.
  • Accessibility checked using Lighthouse and Wave tools.
  • Color contrast verified for readability.
  • SEO basics implemented: meta tags, semantic HTML, and descriptive titles.
  • Performance optimized with clean code and minimal assets.
